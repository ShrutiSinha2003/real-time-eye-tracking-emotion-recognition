# Real-Time Eye Tracking and Emotion Recognition System

## Overview
A multimodal cognitive state analysis system that combines eye movement tracking and emotion fusion to classify focused and distracted states using machine learning.

## Features
- Real-time blink detection  
- Gaze tracking  
- Saccade detection  
- Emotion fusion using DeepFace  
- Cognitive state classification  
- Attention score visualization  
- Gaze heatmap generation  

## Tech Stack
- Python  
- OpenCV  
- MediaPipe  
- DeepFace  
- Scikit-learn  
- Random Forest Classifier  
- Matplotlib  
- Seaborn  

## Results
- Classification Accuracy: **97%**  
- Confusion Matrix:
```text
[[32,1],
 [0,27]]
```

## Applications
- Attention monitoring  
- E-learning engagement analysis  
- Driver distraction detection  
- Human-computer interaction

## How to Run
```bash
pip install -r requirements.txt
python main.py
```
