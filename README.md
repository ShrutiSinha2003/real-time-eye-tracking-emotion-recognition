# Real-Time Eye Tracking and Emotion Recognition System

## Overview
A cognitive state analysis system using OpenCV, MediaPipe, DeepFace, and Random Forest.

## Features
- Blink detection
- Gaze tracking
- Saccade detection
- Emotion fusion
- Attention classification

## Results
- Accuracy: 97%
- Confusion Matrix:
[[32,1],
 [0,27]]
